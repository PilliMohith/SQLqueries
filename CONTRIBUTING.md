**What To Do**

Create a single file containing all your solutions.

Name the file using your name, for example: solutions_mohith.sql.

Answer every question in that file.

Ensure all queries run correctly on the schema below.

Keep SQL formatted and readable.

**How To Contribute**

Fork the repository.

Create a new branch for your solutions.

Add your file in the branch.

Do not edit or replace files submitted by others.

Commit your work and open a pull request when all answers are completed.

Use the pull request template given below.

**Schema To Use**

**1)Students Table**

student_id: Primary Key

roll_no: Unique roll number

name: Student name

age: Student age

email: Unique email

city: City of the student

address: Address (can be NULL)

marks: Total marks

dept_id: Foreign Key referencing departments

created_at: Timestamp

**2)Departments Table**

dept_id: Primary Key

dept_code: Unique department code

dept_name: Department name

**Additional Guidelines**

Use standard SQL syntax.
